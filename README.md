# Setup instruction

Method 1 - Clone this repo directly
## 1.Clone this repo:
```bash
rm -rf package/openwrt-packages
git clone --depth=1 https://github.com/AlphaBotZ/openwrt-packages.git package/openwrt-packages
```
## 2.Pull upstream commits:
```bash
git -C package/openwrt-packages pull
```

## Remove
```bash
rm -rf package/openwrt-packages
```
